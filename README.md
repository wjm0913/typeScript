# typeScript 笔记
> ###  安装 typeScript 命令用于编译成 javaScript
* 安装环境
    > npm install typescript -g 
* 执行拜编译命令
    > tsc index.ts 
* 编译热加载命令
    > tsc -w index.ts
 > ### var  let  const 区别
    1. var 函数级作用域 
    2. let 定义变量 限制变量的作用范围，防止变量的重复定义(相同类型)
    3. const 定义常量 通俗的说 定义死了  不能随意更改
